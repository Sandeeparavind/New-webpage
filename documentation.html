Recipe Management Application

Developed a CRUD application with JWT authentication, enabling users to create, update, and delete recipes. The application features a user-friendly UI for an interactive recipe menu. Utilized the FIREBASE framework for backend and database management.

Application Functionality

  • Post-authentication, users can view, add, delete, and edit recipes.
  • Recipe parameters include name, image URL, description, and ingredients.
  • An integrated shopping list allows users to add and manage ingredients.
  • JWT implementation ensures automatic user login after application closure.
  • Token expiration and automatic logout after one hour.

Data Access and Manipulation

To access and manipulate data:

  • Navigate to the navigation bar and select "Manage" > "Fetch data".
  • Perform desired actions (view, add, delete, or edit recipes).
  • Commit permanent changes to the database by selecting "Manage" > "Save data".
  • Logout of the application when finished.

Additional Features

The application also includes an option to create a new user during the initial setup.

Conclusion

This CRUD application with JWT authentication provides a comprehensive and user-friendly platform for managing recipes, ensuring data integrity and user convenience.
